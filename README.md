# 🚀 Users & Posts Recap App (Next.js + Express)

A full-stack web application built with **Next.js (frontend)** and **Express.js (backend)** to manage users and posts using REST APIs.

# ✨ Features

## 👤 Users
- Get all users
- Add new user (POST API)
- Active / Inactive status

## 📝 Posts
- Get all posts
- Add new post (POST API)
- User-based post system

## 🌐 API Integration
- REST API structure
- JSON data handling
- Frontend ↔ Backend connection

---

# 📡 API Endpoints

## Users

GET /users
POST /users


## Posts

GET /posts
POST /posts

# 🚀 How to Run Locally

## Backend

npm install
npm start


Runs on:

http://localhost:5000


---

## Frontend

npm install
npm run dev


Runs on:

http://localhost:3000
